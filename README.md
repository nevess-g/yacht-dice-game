# yacht-dice-game
Este repositório integra o Exercício Programa 2, ofertado pelo Insper no semestre 2026.1 aos cursos de Engenharia. O objetivo do jogo é que o jogador role os dados e tente obter a maior pontuação possível, podendo escolher quais dos dados manterá ou rolará novamente, de modo a formar combinações específicas.
